README — Zamat
===============

Overview
--------
Zamat is a web site for a traditional Mauritanian board game (checkers/draughts), focused on preserving the authentic
rules and providing a smooth browser-based experience. The project is published on GitHub and used to deploy the
website online.

Features
--------
- Player vs Computer: configurable AI via in-game settings.
- Online Player vs Player: invitations and match synchronization.
- Save & Resume: store game state in the browser and continue later.
- Move / Event log: clear tracking of gameplay.
- Multiple languages: Arabic, English, French.

Website pages
-------------
- Home: index.html
- Rules: rules.html
- Privacy Policy: privacy.html
- About: about.html
- Contact: contact.html

Technology (high level)
-----------------------
- Gameplay UI: 2D Canvas with an optional 3D board view.
- AI with advanced options (thinking time / weights / strategies).
- Online services: Firebase (Realtime Database + Auth) for online play and synchronization.
- Libraries: onnxruntime-web, three.js, Firebase Web SDK.

Privacy & data
--------------
The website processes data necessary for online play and user experience (e.g., nickname, presence, match/session
data for synchronization, and browser preferences). See:
privacy.html
