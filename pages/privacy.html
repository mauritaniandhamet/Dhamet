نسخة ويب متقدمة من اللعبة الموريتانية التقليدية ظامت (وتُنطق أيضًا: Dhamet, Zamma, Zamet, Zamat). وهي نوع من ألعاب الداما العالمية (checkers/draughts).

سياسة الخصوصية

آخر تحديث: 13 يناير 2026

1) البيانات التي يعالجها التطبيق

1.1) بيانات اللعب لتدريب الذكاء الاصطناعي

يسجّل التطبيق بيانات من المباريات بهدف تدريب وتحسين نموذج الذكاء الاصطناعي، وتشمل عادةً:

  • تسلسل النقلات وحالات الرقعة اللازمة لبناء بيانات التدريب.
  • نتيجة المباراة ومدة اللعب وعدد النقلات.
  • مستوى الصعوبة/وضع اللعب عند توفره ضمن بيانات المباراة.

تُحفظ هذه البيانات بصيغة لا تتطلب اسمًا حقيقيًا للمستخدم.

1.2) بيانات اللعب عبر الإنترنت (Firebase)

يعالج التطبيق بيانات لازمة لتشغيل اللعب عبر الإنترنت عبر Firebase، وتشمل:

  • اسمًا مستعارًا تختاره للظهور للاعبين الآخرين.
  • حالة التواجد (متصل/في مباراة) لتسهيل العثور على اللاعبين.
  • بيانات الجلسة والمباراة (حالة الرقعة، النقلات، الدور، وسجل المباراة) لضمان المزامنة بين الطرفين.

1.3) بيانات محلية على جهازك

يحفظ التطبيق داخل المتصفح بيانات لتحسين التجربة، مثل:

  • اللغة والإعدادات العامة.
  • حفظ/استئناف المباراة على جهازك.
مساحة إعلانية (مربع داخل المحتوى)

1.4) بيانات الإعلانات

تعرض الخدمة إعلانات، ويعالج مزود الإعلانات بيانات تقنية لازمة لعرض الإعلان وقياس الأداء ومنع الاحتيال، مثل ملفات تعريف الارتباط، ومعرّفات المتصفح، ومعلومات الجهاز و/أو عنوان IP لأغراض تشغيل الإعلانات.

1.5) بيانات التواصل

عند مراسلتنا، يعالج التطبيق بيانات الرسالة ومعلومات الاتصال التي تزوّدنا بها للرد ومعالجة الطلب.

2) استخدام البيانات

تُستخدم البيانات للأغراض التالية:

  • تشغيل اللعبة وميزات الحفظ والإعدادات.
  • تمكين اللعب عبر الإنترنت ومزامنة المباريات وإدارة الدعوات.
  • تدريب وتحسين نموذج الذكاء الاصطناعي ورفع جودة اللعب.
  • عرض الإعلانات وقياس أدائها وحماية المنصة من الاحتيال.
  • تحسين الاستقرار واكتشاف الأعطال ومعالجة الأخطاء.
  • الرد على طلبات الدعم والتواصل.

3) الخدمات الخارجية (Firebase وGitHub والإعلانات)

  • Firebase: تُستخدم لتشغيل ميزات اللعب عبر الإنترنت وتخزين بيانات الجلسات والمزامنة وما يلزم لها.
  • GitHub: يستضيف مستودع المشروع، وتُستخدم بنيته (مثل GitHub Actions) ضمن سير عمل التطوير والتدريب وإصدار تحديثات نموذج الذكاء الاصطناعي.
  • مزودو الإعلانات: يعالجون البيانات اللازمة لعرض الإعلانات وقياسها وفق سياساتهم.

4) مشاركة البيانات

لا يبيع التطبيق البيانات. تُشارك البيانات فقط مع:

  • Firebase لتشغيل اللعب عبر الإنترنت والبنية التحتية ذات الصلة.
  • مزودي الإعلانات لتقديم الإعلانات وقياسها ومنع الاحتيال.
  • مزودي الاستضافة والأمان عند الحاجة لتشغيل الخدمة وحمايتها.

5) الاحتفاظ بالبيانات

  • البيانات المحلية تبقى على جهازك حتى تقوم بحذف بيانات الموقع من المتصفح.
  • بيانات اللعب عبر الإنترنت تبقى بالقدر اللازم لتشغيل الجلسات والمزامنة.
  • بيانات التدريب تُحفظ ضمن مجموعات بيانات لتحسين نموذج الذكاء الاصطناعي.

6) خياراتك

  • يمكنك استخدام اسم مستعار غير معرّف شخصيًا.
  • يمكنك حذف البيانات المحلية عبر مسح بيانات الموقع من المتصفح.
  • يمكنك التحكم في ملفات تعريف الارتباط/الإعلانات من إعدادات المتصفح وإعدادات مزود الإعلانات.

7) الأمان

يطبق المشروع إجراءات أمان معقولة لحماية البيانات، مع الإقرار بأن بيئات الإنترنت لا تضمن حماية مطلقة.

8) التواصل

للاستفسارات المتعلقة بالخصوصية:

  • عبر صفحة «تواصل معنا» داخل الموقع.

جميع الحقوق محفوظة.

هذا المشروع، بما في ذلك الشفرة المصدرية، والأصول (Assets)، والمحتوى، والنماذج، والوثائق (ويشار إليها مجتمعةً بـ"البرنامج") هو ملكية خاصة ويعود لـ The Mauritanian Software Lab ("المرخِّص").

1) ليس ترخيصًا مفتوح المصدر

البرنامج ليس مفتوح المصدر. لا تُمنح أي حقوق للاستخدام أو النسخ أو التعديل أو الدمج أو النشر أو التوزيع أو الترخيص من الباطن أو البيع أو أي استغلال آخر، إلا وفق ما هو منصوص عليه صراحةً في هذا الترخيص.

2) السماح المحدود: الوصول عبر الإنترنت فقط

مع مراعاة الشروط أدناه، يُسمح لك بالوصول إلى الموقع الرسمي (النسخة المنشورة عبر الإنترنت) واستخدامه فقط:

  • عبر الروابط الرسمية للمرخِّص.
  • للاستخدام الشخصي فقط.

لا يُمنح أي حق لتشغيل/استضافة/نشر نسخة أخرى أو إعادة نشر البرنامج خارج النشر الرسمي عبر الإنترنت.

3) القيود (لا نسخ/لا تنزيل/لا تعديل/لا أعمال مشتقة)

ما لم تحصل على إذن كتابي مسبق من المرخِّص، فلا يجوز لك كليًا أو جزئيًا:

  • نسخ أو تنزيل أو استخراج أو إعادة إنتاج أي جزء من البرنامج.
  • التعديل أو التكييف أو الترجمة أو إنشاء أعمال مشتقة.
  • إعادة التوزيع أو النشر أو الترخيص من الباطن أو البيع أو التأجير أو الاستغلال التجاري.
  • تشغيل أو نشر أو استضافة أو عكس (Mirror) البرنامج خارج النشر الرسمي عبر الإنترنت.
  • الهندسة العكسية أو التفكيك أو محاولة استخراج الشفرة/النماذج، إلا بالقدر الذي يمنعه القانون من تقييد ذلك.

4) صلاحيات GitHub المطلوبة (داخل خدمة GitHub)

إذا كان المستودع عامًا (Public) على GitHub، فقد تسمح شروط GitHub لمستخدمي GitHub بعرض المستودع واستخدام وظيفة "Fork" داخل خدمة GitHub (أي إنشاء نسخة داخل GitHub في حساب/منظمة أخرى) بما يلزم لتقديم الخدمة. هذه الصلاحيات منصّة-مقيدة داخل GitHub ولا تمنح حقوقًا أوسع لاستخدام أو تشغيل أو تعديل أو توزيع البرنامج خارج ما تتيحه GitHub وظيفيًا ووفق شروطها.

5) مكوّنات الطرف الثالث

قد يتضمن البرنامج أو يعتمد على مكتبات/خدمات طرف ثالث تخضع لتراخيص وشروط خاصة بها، وتُطبق تلك الشروط على مكوّناتها بشكل مستقل.

6) الإنهاء

يظل هذا الترخيص نافذًا حتى يتم إنهاؤه. يُنهى تلقائيًا عند مخالفة أي شرط. عند الإنهاء يجب التوقف عن أي استخدام للبرنامج.

7) إخلاء المسؤولية عن الضمانات

يُقدّم البرنامج "كما هو" دون أي ضمانات صريحة أو ضمنية، بما في ذلك (على سبيل المثال لا الحصر) ضمانات القابلية للتسويق، والملاءمة لغرض معين، وعدم الانتهاك.

8) تحديد المسؤولية

إلى أقصى حد يسمح به القانون، لا يتحمل المرخِّص أي مسؤولية عن أي مطالبات أو أضرار أو مسؤوليات أخرى تنشأ عن أو ترتبط بالبرنامج أو استخدامه.

9) التواصل

للدعم أو الاستفسارات أو طلب الإذن: عبر صفحة "تواصل معنا" المنشورة مع الموقع. لا يتم اعتماد البريد الإلكتروني أو الهاتف كقناة رسمية.
